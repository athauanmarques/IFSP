Lição 16: Novas tags e atributos

Nesta lição, você vai entender obter uma compreensão profunda sobre marcas e atributos - e começa a aprender alguns novos.

Block e Inline

A maioria dos elementos HTML ou são em nível de bloco elementos ou de nível de linha elementos. Elementos nível de bloco apenas significar que esses são os elementos que definem a estrutura do site ou eles são elementos que contêm a maior parte do conteúdo. Geralmente, eles começam em uma nova linha. Exemplos: <h1> , <p> , <table> , <ul> .

Inline elementos são encontrados dentro de frases normalmente e não indicaria que o conteúdo deve começar em uma nova linha. Exemplos: <b> , <img> , <em> .

Elementos estruturais

Estruturais elementos são todos os elementos nível de bloco e ajudar a identificar o tipo de conteúdo e da área na página onde o conteúdo será colocado.

Um novo elemento estrutural é <article> . Ele foi criado por qualquer conteúdo que você considerar completo e abrangente sobre o seu próprio: uma entrada de blog, um artigo de revista, um livro, uma tese, um comentário. Pode ter muitas seções (capítulos, partes) para ele.

O segundo elemento estrutural que você precisa saber é uma <section> e você usá-lo para uma seção de um artigo. É um agrupamento lógico de informações e o melhor teste é saber se você precisa de uma legenda para ele. Se você quiser colocar um título em um subconjunto de pontos, então você pode usar <section> .

Exemplo 1: o artigo com uma secção

	
	  <Body>
	    <Article> 		
	      Entry <h1> Meu Blog </ h1> 		
	      <P> Este blog é o meu primeiro. </ P>
	      <Section> 			
	        <H2> O que eu aprendi </ h2>
	        <P> HTML5 pode ter artigos e seções. </ P>
	      </ Section>
	    </ Article> 
	  </ Body>

cabeçalho e rodapé são os próximos novos elementos estruturais que você precisa saber. Eles podem conter todos os tipos de coisas, mas isso seria onde, por exemplo, iria colocar o título de sua página (cabeçalho) ou suas informações de direitos autorais (rodapé).

Você pode realmente ter muitos cabeçalhos e rodapés () em uma página. Você pode ter um cabeçalho para a página inteira e outro cabeçalho para o artigo. Para a página, você deve incluir o nome da página web. Para o artigo, o cabeçalho pode ser o título do artigo, bem como o autor e data.

Classes

As aulas são um pouco ferramenta útil que atribui a informação a um elemento para que você possa fazer mais coisas com alguns ou todos os elementos que pertencem a uma determinada classe.

Por exemplo, se você queria alguns de seus artigos por ter um limite especial ou fonte, você pode identificar os artigos com uma classe.

	
	<Body>
	  <Artigo class = "fancy" >	
	    Entry <h1> Meu Blog </ h1> 		
	    <P> Este blog é o meu primeiro. </ P>
	  </ Article>
	</ Body>

Mantenha aulas em mente que você começar a aprender CSS - a peça que permite formatar sua página.

IDs

IDs, como aulas, são metadados que você conecta ao elementos. IDs são identificadores exclusivos para um determinado elemento. Eles devem sempre começar com uma letra, mas você pode adicionar números depois.

Você poderia definir um ID em um elemento, se você precisa fazer alguma coisa em particular com esse elemento, como executar algum JavaScript nele ou até mesmo definir algum estilo exclusivo sobre ele.

	
	<Button id = "Play1" onclick = "play ();">
	  <Image src = "play_button.png" />
	</ Button>

Você é mais propensos a usar JavaScript em itens que têm interatividade. Neste caso, o nosso botão de reprodução precisa fazer algo extravagante. Um pedaço de código Javascript pode recuperar um elemento pelo seu id específico de "Play1" de modo que quando alguém clica no botão (onclick), ele vai jogar o que for necessário - definido pela Javascript.

Os detalhes do Javascript são um pouco avançada demais para este tutorial, mas você precisa entender que adicionar o ID torna-se um componente essencial de ser capaz de adicionar interatividade ao seu site.

Atributos globais

Há 15 atributos globais - atributos que você pode adicionar a qualquer elemento. O primeiro e mais importante é o atributo ID, que já explicamos. Você adicioná-lo a qualquer elemento que você precisa para controlar especialmente.

Outros atributos legais que são novos para HTML5 incluem contenteditable , o que torna conteúdos editáveis ​​no navegador, de um elemento arrastáveis ​​, que permite que um usuário arraste esse elemento ao redor, e traduzir , o que pode controlar se o conteúdo do elemento é traduzido (ou não traduzido!) quando alguém acessa-lo através de um browser configurado para um idioma diferente.

Há também todo um conjunto de atributos específicos para formas que vamos olhar na próxima lição.

Vinculação

Você vai se lembrar que as ligações geralmente parecido com este:

	
	</a> <a href="my_second_page.html"> Segunda Página

Bem, e se você quiser que o link para abrir uma nova janela do navegador (para que as pessoas não navegar longe de seu site)?

O alvo atributo resolve esse problema. O alvo atributo especifica o que acontece no navegador quando alguém clica no link. Para abrir uma nova janela ou aba, use o _blank valor.

	
	<A target = "_ blank" href = "my_second_page.html"> Segundo </a> Página

Tente você mesmo, ligando uma página para uma segunda página que você criou. Quando você abre a página com o link no navegador, contanto que você tem target = "_ blank" set, ele vai abrir uma nova aba ou janela.

Você também pode identificar o tipo de link que você está usando quando você especificar o rel atributo. Você pode definir isso em uma <a> elemento, um <link> elemento, ou um <area> elemento (a maneira de colocar hotspots em uma imagem). Isso responde a pergunta de por que você está apontando para outra página.

Existem duas categorias de vinculação: links para recursos externos que aumentam o documento e hyperlinks atual, que só liga duas páginas juntas.

A <link> para um recurso externo seria rel = "stylesheet" , o que você vai aprender muito mais sobre em tutorial CSS. A maioria dos outros tipos de ligação seria apenas um hyperlink regular.

Você pode estar se perguntando um pouco sobre a diferença entre o <a> elemento ea <link> elemento. Ambos tomam o atributo rel, mas o <a> elemento tem algo clicável dentro dele (a palavra, gráfico, ou o botão), enquanto que o elemento de ligação não exibe nada na página - ele é usado principalmente para conectar-se a recursos, como uma folha de estilo.

Há uma série de valores permitidos, mas alguns dos mais notáveis ​​são autor , alternativo , ajuda , próximo , anterior , e busca .

Se o link acima estava conectando a primeira página de um artigo para a segunda página de um artigo, você usaria o rel atributo com o seguinte valor.

	
	<a rel="next" href="my_second_page.html"> </a> Segunda Página

Se eu estava ligando para bio ou o site do autor, eu usaria o autor valor em vez disso. Se eu estava ligando para algum documento de ajuda, gostaria de colocar na ajuda valor.
