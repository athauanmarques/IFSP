Os operadores de comparação

Muitas vezes você precisa comparar diferentes valores e fazer o seu programa JavaScript tomar decisões em função destas comparações.

Por exemplo, você está codificando um script JavaScript para um aplicativo de carrinho de compras. Se o total a ser pago for maior ou igual a 50 dólares aplicar um desconto de 5%, se for menor ou igual a 50 dólares não se aplica 5% de desconto . Não seja impaciente, você vai aprender como codificar este tipo de condições na próxima lição.

É aqui que os operadores de comparação, como igual, menor que , etc entram em cena. Aqui abaixo estão listados todos os operadores de comparação para sua referência.

Perguntas, perguntas, perguntas

As tabelas acima são auto-explicativas, exceto para os seguintes 2 perguntas:

  1. Quando você fala sobre ===, o que se entende por igualdade de valor e tipo ?
  2. Qual é a diferença entre (=) , (==) e (===) ?

Resposta à pergunta 1.

Os valores são os dados específicos, diretamente em suas declarações de JavaScript ou contida em variáveis ​​JavaScript. Por exemplo: 

	  
	    var preço = 5;

No trecho de código acima, a variável preço tem o valor 5 .

Qual é o tipo ?

O tipo, ou mais precisamente o tipo de dados , é o caminho JavaScript classifica os dados. Você me deparei com 2 tipos de dados, isto é, número e string (texto) . Um tipo de dados terceiro é Boolean , ou seja, as afirmações verdadeiras e falsas .

Portanto, quando você comparar 2 valores usando (===), os 2 valores são comparados com base tanto o seu valor e seu tipo de dados : 

	  
	    var firstNum = 4;    
	    var secondNum = 4;    
	    // Isso é verdade: ambos os valores são 4    
	    // E ambos os valores são de número do tipo de    
	    firstNum === secondNum;    
	    // Vamos usar um tipo de dados.   Uma seqüência de caracteres usa '.    
	    var stringNum = '4 ';    
	    // Agora === é falsa: 4 e '4 'são tipos diferentes    
	    firstNum === stringNum;

Resposta à pergunta 2.

O operador (=) é usado para atribuir ou dar um valor a uma variável. NÃO é um sinal de igualdade .

Os operadores == e === são operadores de igualdade. O == compara os valores apenas, o === compara os valores e tipos de dados.
