Lição 20: Efeitos de animação com jQuery

Traduzido de Maria Antonietta Perna

jQuery é muito rico em funções prontas para a adição de efeitos especiais na sua página web. Não é difícil se deixar levar por ele.

Aqui, nós estamos indo para explorar os efeitos mais comumente usados ​​do jQuery. Mais uma vez, a recomendação é a de integrar esta lição com uma visita ao site jQuery para mais exemplos de código e explicações detalhadas.

Nesta lição, você vai aprender como:

  • adicionar efeitos manipulando dinamicamente estilos ;
  • usar os métodos show() / hide() / toggle() ;
  • usar os métodos fadeIn() / fadeOut() / fadeToggle() ;
  • usar os métodos slideUp() / slideDown() / slideToggle() .

Neste processo, você vai construir um simples menu de deslizamento e obter mais experiência com a nova abordagem jQuery.

Como faço para adicionar e remover classes CSS com jQuery?

jQuery permite manipular os estilos CSS no código com $.css (styleName, styleValue) . Ele também permite que você adicione e remova classes CSS em relação aos elementos DOM.

Vamos nos concentrar em aplicar e remover classes CSS com $.addClass(className) , $.removeClass (className) , e $.toggleClass (className) . Esta abordagem utiliza as declarações de classe CSS armazenados tanto no <head> da sua página HTML ou em um arquivo separado, portanto, é mais adequado para melhores práticas correntes na web design.

Se você está curioso sobre $.css (), recomendo visitar http://api.jquery.com/css/ para mais detalhes.

Suponha que você queira substituir a classe CSS bigDiv com a classe CSS smallDiv quando o usuário clica no interior do elemento div. Isto é como você poderia fazer a tarefa com jQuery: 

		
	 $(document).ready (function ()  
		 {  
		 	// Armazenar uma referência ao div em uma variável usando seu id:  
		 	var myDiv = $ ("# myDiv ');  
		 	// Anexar um manipulador de clique para manipular as classes CSS:  
		 	myDiv.click (function ()  
		 	{  
		 		// Remove a classe CSS existente e substituí-lo por um novo.  
		 		// Uma vez que o elemento é o mesmo, da cadeia de ambos os métodos:  
		 		myDiv. removeClass ('bigDiv') addClass ('smallDiv').   ;  
		 	});  
		 });

E quanto a alternância entre a CSS aulas bigDiv e smallDiv clicando dentro da div? É fácil: jQuery tem este método pouco prático para você - $ .toggleClass () . Assumindo a classe CSS bigDiv é hard-coded no atributo de classe do elemento div, você pode alternar a classe CSS smallDiv usando jQuery, assim: 

		
		 $ (Document) .ready (function ()  
		 {  
		 var myDiv = $ ("# myDiv ');  
		 myDiv.click (function ()  
		 {  
		 . myDiv   toggleClass ('smallDiv')   ;  
		 });  
		 });

Agora, continue clicando no interior do elemento div, e você vai vê-lo ficando menor e maior a cada clique do mouse.

Como faço para usar o programa de jQuery () / hide ()?

Se você quer mostrar elementos HTML / esconder em uma página, você pode fazê-lo facilmente com jQuery show () / hide () . Você também pode modular a maneira como os elementos são revelados ou escondidos, adicionando os "lento" ou "rápido" argumentos, ou até mesmo o número de milissegundos que você gostaria a transição para durar.

Vamos dizer que você tem um elemento de parágrafo dentro de uma div com um id myDiv . Você quer que o parágrafo a desaparecer lentamente quando o usuário clica no interior do elemento div. Veja como você pode fazer isso com jQuery hide () . 

		
		 $ (Document) .ready (function ()  
		 {  
		 var myDiv = $ ("# myDiv ');  
		 myDiv.click (function ()  
		 {  
		 // Substitua "lento" com 2000 por uma transição mais lenta.  
		 $ ('P')   .hide ("lento")   ;  
		 });  
		 });

Clicando no interior dos resultados de elementos div no parágrafo lentamente desaparecendo da vista. jQuery show () funciona da mesma maneira. Deixo-vos a experimentar com ele em seu próprio país.

E se você quiser alternar entre mostrar / ocultar transições como o usuário clica no interior do elemento div? Não tem problema, jQuery oferece a $ .toggle () método. Para vê-lo em ação, basta substituir hide () com alternância () no exemplo anterior, assim. 

		
		 $ (Document) .ready (function ()  
		 {  
		 var myDiv = $ ("# myDiv ');  
		 myDiv.click (function ()  
		 {  
		 $ ('P')   .toggle ("lento")   ;  
		 });  
		 });

Como eu faço elementos DOM fade in e out da página?

Para um efeito de esmaecimento mais sofisticado, use $ .fadeIn () / $. fadeOut () / $. fadeToggle () . Você pode usar estes métodos da mesma forma que você usou $ .show () / $. Hide () / $. Alternar () nos exemplos anteriores.

Aqui está o parágrafo a partir da alternância exemplo de código anterior entre visível e escondido usando $ .fadeToggle (). 

		
		 $ (Document) .ready (function ()  
		 {  
		var myDiv = $ ("# myDiv ');  
		 myDiv.click (function ()  
		 {  
		 $ ('P')   .fadeToggle ("lento")   ;  
		 });  
		 });

O efeito é muito semelhante à da .toggle $ ('retardar'). Basta experimentá-lo.

Como eu faço elementos DOM deslizar para cima e para baixo com jQuery?

Você pode facilmente fazer elementos DOM aparecem e desaparecem com um efeito de deslizamento, utilizando $ .slideUp () / $. slideDown () / $. slideToggle () . Está aplicar estes métodos semelhante aos anteriores.

Por exemplo, digamos que você deseja para deslizar o elemento de parágrafo do exemplo anterior para cima e para baixo como o usuário clica no interior do elemento div. Veja como você iria fazer isso com $ .slideToggle () . 

		
		 $ (Document) .ready (function ()  
		 {  
		 var myDiv = $ ("# myDiv ');  
		 myDiv.click (function ()  
		 {  
		 $ ('P')   .slideToggle ("lento")   ;  
		 });  
		 });

Agora continue clicando no interior do elemento div e ver o parágrafo dentro da div deslizando para cima e para baixo.

Eu recomendo que você experimentar com as amostras de código acima, tanto quanto você puder antes de seguir em frente com o próximo desafio.

Experimente: Menu deslizante

Aqui estamos no nosso jQuery experimentar exercício. Você vai construir um menu simples deslizamento semelhante ao que você já deve ter visto em muitos sites.

O usuário move o mouse sobre um item do menu principal e uma lista de itens do submenu desliza para baixo. À medida que o usuário move o mouse para longe do item de menu, os seus itens do sub-menu deslize de volta para longe da vista.

Vamos começar a partir da página HTML: 

		  
		<! DOCTYPE html>   
		<Html>   
		<Head>   
		<Title> Lição 20: jQuery deslizante menu </ title>  
		 <Script type = "text / javascript" src = "http://code.jquery.com/jquery-1.7.min.js"> </ script>  
		 <Script type = "text / javascript" src = "lesson20_ex.js"> </ script>  
		</ Head>   
		<Body>   
		<H1> Lição 20: jQuery Sliding Menu </ h1>  
		 <Div id = "myDiv">  
		 <Ul class = "outerList">   
		<Li> <a href="#"> Menu Item 1 </a> </ li>   
		<Li> <a item href="#"> Menu 2 ↓ </a>  
		 <Div>  
		 <ul class = "InnerList">   
		<Li> <a href="#"> sub-menu 1 </a> </ li>   
		<Li> <a href="#"> item Sub-Menu 2 </a> </ li>   
		<Li> <a href="#"> sub-menu 3 </a> </ li>   
		</ Ul>   
		</ Div>  
		</ Li>   
		<Li> <a href="#"> Menu Item 3 </a> </ li>   
		</ Ul>  
		</ Div>  
		</ Body>   
		</ Html>

A página HTML acima tem uma referência a uma cópia CDN servida do jQuery e para um arquivo JavaScript externo no <head> seção.

A seção <body> tem um elemento div com um id myDiv que contém uma lista com 3 itens do menu principal e 3 itens de submenu aninhados dentro do segundo item do menu principal. Os itens do sub-menu estão contidos dentro de um elemento div.

Estilo de seu menu para que os itens do menu principal são alinhados horizontalmente e os itens do sub-menu aninhados são empilhados verticalmente sob o seu elemento li pai (dê uma olhada na página de exemplo , se você quiser).

Queremos que o elemento div que envolve os itens do sub-menu para deslizar para baixo quando o usuário move o mouse sobre o seu elemento li pai.

Prepare a sua lesson24_ex.js arquivo e escrever o seguinte código. 

		
		 $ (Document) .ready (function ()  
		 {  
		 // Recuperar os subitens do menu (div criança elemento do elemento da lista)  
		 // Com os seletores jQuery poderosas  
		 // E armazená-los em uma variável  
		 var = subitens $ ('div ul li');  
		 // Recuperar os itens do menu principal que  
		 // Tem os subitens recuperados como crianças.  
		 //   Observe os .Tem acessíveis () método   :  
		 var mainItems =   $ ('ul li') tem (subitens).   ;  
		 // Esconder todos os subitens no carregamento da página  
		 subitens   .Hide ()   ;  
		 // Anexar a função .hover () para o principal  
		 // itens de menu:  
		 $ (MainItems) .hover (function ()  
		 {  
		 // Aplique .slideToggle () para o sub-menu  
		 subItems.slideToggle ('fast')   ;  
		 });  
		 });

Passe o mouse sobre o segundo item de menu e se todos correr como esperado, você verá o sub-menu deslizante para baixo. Levou-nos apenas algumas linhas de código para realizar um efeito de animação que, em matéria-JavaScript teria nos fez suar um pouco.
