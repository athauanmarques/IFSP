Introdução

Por Maria Antonietta Perna

JavaScript lhe dá a liberdade para adicionar interatividade e capacidade de resposta para suas páginas web.

O objetivo deste tutorial é fornecer-lhe com uma introdução, completa acessíveis a JavaScript usando explicações mal-humorado e tarefas práticas para experimentar desde o início.

Nenhum conhecimento prévio do JavaScript é assumido, mas porque o JavaScript fica dentro e manipula páginas da Web, a fim de ser capaz de seguir adiante, você já deve estar familiarizado com HTML e CSS . Se você é novo em um ou ambos, é aconselhável a percorrer o nosso HTML e CSS tutoriais em primeiro lugar.

JavaScript é leve, fácil de aprender, linguagem de script. É usado em quase todos os site para responder às ações do usuário, validar formulários web, detectar o suporte ao navegador, e muito mais.

JavaScript é uma linguagem de programação web, ou seja, uma linguagem que permite que você, o designer do seu site, para controlar a forma como uma página web se comporta . Isso faz com JavaScript crucialmente diferente de HTML, a linguagem que dá estrutura aos seus documentos web, e CSS, a linguagem que controla a aparência das páginas da web.

Se você conhece outras linguagens de programação como PHP , a maioria dos conceitos de programação e sintaxe JavaScript básico, vai soar bem familiar para você. No entanto, se este não for o caso, não se preocupe: por acompanhando e fazendo experiências com o código , no final deste hands-on tutorial você será capaz de enfeitar suas páginas web estáticas com efeitos divertidos e receptividade fantástica para a alegria dos visitantes do seu site.

Se precisar de ajuda ao longo do caminho, não se esqueça de ligar para os nossos fóruns . Este é o lugar onde você se encontra com os verdadeiros especialistas que estão dispostos e prontos para oferecer dicas, sugestões e conselhos.

O que é necessário?

Para dar vida a suas páginas web com JavaScript tudo que você precisa é um editor de texto e uma actualização, browser compatível com o padrão de sua escolha, como o Internet Explorer 9, Firefox, Chrome, Safari ou Opera, para citar apenas os mais populares .

Básicos editores de texto livres que vendem com o seu sistema operacional, como o Bloco de Notas (no Windows) e editar texto (no Mac) vai fazer muito bem. No entanto, editores de texto que oferecem suporte linguagens de programação, tais como a livre Notepad + + (para usuários do Windows) e TextWrangler (para usuários Mac), pode ser um ótimo recurso quando arar através de mais do que algumas linhas de código.
