Lição 2: O Primeiro JavaScript

Nesta lição você vai aprender:

  • como incorporar o seu JavaScript na página HTML;
  • como referênciar o seu JavaScript a partir de um arquivo separado;
  • como comentar seu código JavaScript;
  • como criar sua primeira página web JavaScript-powered.

O HTML

Para inserir um script JavaScript em uma página HTML, você usa o <script> ... </ Script>.

Vamos começar com uma página de HTML básico, como esta: 

	     
	  <! DOCTYPE html>    
	  <html>    
	  <head>    
	  <title> Minha  primeira página JavaScript </ title>   
   
	  </ Head>    
	  <body>   
   
	  </ Body>    
	  </ Html>

O script JavaScript é inserido tanto na página HTML em si ou em um arquivo separado.

JavaScript embutido na página HTML

A tag <script> e seu atributo tipo dizem ao navegador: "Hey, browser Há um script chegando, e é um script JavaScript!".

É possível fazer isso, quer na secção <head>, como se segue: 

	     
	  <! DOCTYPE html>    
	  <html>    
	  <head>    
	  <title> Minha  primeira página JavaScript </ title>   
		  
       <script type="text/javascript">  
	   // código JavaScript vai aqui  
       </ script>   
		   
	  </ Head>    
	  <body>   
   
	  </ Body>    
	  </ Html>

Ou na parte inferior do documento antes do fechamento </ body> tag, assim: 

	   
	  <! DOCTYPE html>    
	  <html>    
	  <head>    
	  <title> Minha primeira página JavaScript </ title>    
	  </ Head>    
	  <body>   
          
       <script type="text/javascript">  
       // código JavaScript vai aqui  
       </ script>   
		   
	  </ Body>    
	  </ Html>

Se você está se perguntando se é melhor colocar a sua marca <script> no <head> ou o <body>, então você não está sozinho. Isto sempre se resume a sua preferência pessoal.

Comments, comentários, comentários

Uma última coisa a notar sobre ambos os trechos de código acima são as duas barras // antes do texto "código JavaScript vai aqui". Isto é como você comenta uma linha de código JavaScript .

Quando um comentário se estende por mais de mais de uma linha, você pode usar / * Comentário vai aqui * / para delimitar um comentário, assim como você faz em uma folha de estilo. Veja como isto é feito: 

	   
	  <! DOCTYPE html>    
	  <html>    
	  <head>    
	  <title> Minha primeira página JavaScript </ title>    
	  </ Head>    
	  <body>   
          
       <script type="text/javascript">  
	   / * código JavaScript  
	   vai aqui * /  
       </ script>   
		   
	  </ Body>    
	  </ Html>

Quando o interpretador JavaScript do seu navegador se depara com '//' ou '/ ** /', ele simplesmente ignora qualquer texto está colocado entre os dois. Use comentários em seu código para lembrar no futuro do que seu código foi projetado para fazer.

Inserir JavaScript em um arquivo separado

Se o seu script é mais longo do que algumas linhas, ou você precisa aplicar o mesmo código para diversas páginas em seu site, então é melhor empacotar seu código JavaScript em um arquivo separado.

Isto, facilitará a manutenção e reutilização de seus scripts.

Veja como isto é feito: 

	   
	  <! DOCTYPE html>    
	  <html>    
	  <head>    
	  <title> Minha primeira página JavaScript </ title>    
      <script type="text/javascript" src="yourjavascript.js"> </ script>    
	  </ Head>    
	  <body>   
           
	  </ Body>    
	  </ Html>

Como você pode ver, o <script> referencia um arquivo externo, "yourjavascript.js" que tem a extensão . js e contém o script que coloca a magia em sua página web.

Sua primeira página de JavaScript: Olá Mundo

Sem mais delongas, vamos ver se funciona JavaScript no seu navegador.

Experimente: JavaScript embutido em uma página HTML

Entre o <script> e </ script> tanto na <head> quanto no <body> de seu documento HTML, adicione a seguinte linha de código, logo após o comentário: 

	   
	  <! DOCTYPE html>    
	  <html>    
	  <head>    
	  <title> Minha página primeiro JavaScript </ title>    
	  </ Head>    
	  <body>   
          
	   <script type="text/javascript">  
	   // código JavaScript vai aqui  
	   alert ('Olá Mundo!');  
       </ script>   
		   
	  </ Body>    
	  </ Html>

Esta é a sua primeira declaração JavaScript, isto é, você acabou instruído sua página web para fazer algo. Não se preocupe com o código ainda, estaremos voltando para o comando alert () de novo nas aulas seguintes.

Basta notar o ponto e vírgula (;) no final da declaração. Isso é importante: ele diz ao interpretador de JavaScript que a declaração está terminada e que vem a seguir é uma declaração diferente.

Agora salve o seu trabalho e execute a página no seu navegador favorito. Você verá uma caixa de alerta aparecendo no carregamento da página. Esta é a aparência da página no Firefox:

Se a sua caixa de alerta não está aparecendo, então verifique se você digitou o comando JavaScript exatamente como está no código de exemplo.

Verifique se o seu <script> ... </ script> estão lá, que o texto entre colchetes está entre aspas ("") , e que existe um ponto e vírgula (;) no final da declaração . Em seguida, tente novamente.

Experimente: JavaScript em um arquivo separado

Criar um novo documento no seu editor de texto e salvá-lo como "helloworld.js". Importante: a extensão do arquivo tem que ser js. (esta é a extensão de arquivo apropriada JavaScript).

No novo documento, cole no comando JavaScript do exemplo anterior ( não há necessidade de digitar o <script> ... </ script> marcas aqui): 

	
	   alert ('Olá Mundo!');

Agora, volte para a página HTML, exclua o código JavaScript anterior, e adicione o <script> ... </ Script> marcas na seção <head> da página com uma referência ao arquivo JavaScript helloworld.js, assim: 

 	  <! DOCTYPE html>    
	  <html>    
	  <head>    
	  <title> Minha página primeiro JavaScript </ title>    
	  <script type="text/javascript" src="helloworld.js"> </ script>    
	  </ Head>    
	  <body>   
           
	  </ Body>    
	  </ Html>

Feche todos os seus documentos e execute a página HTML no navegador. Você deve ver a caixa de aparecendo como no exemplo anterior.

Se o código não funcionar, verifique se o caminho para o arquivo JavaScript está correto, a digitação está idêntica ao exemplo, e verifique se você adicionou um fechamento </ script> tag. Em helloworld.js certifique-se de seu comando JavaScript é digitado exatamente o mesmo que no código de exemplo acima, tente novamente.
