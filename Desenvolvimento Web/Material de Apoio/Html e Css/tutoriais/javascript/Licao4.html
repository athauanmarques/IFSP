Lição 4: Variáveis ​​e Constantes

Por Maria Antonietta Perna

Variáveis ​​e constantes são como recipientes onde você armazena os dados e valores para o processamento de JavaScript.

A diferença entre uma variável e uma constante é esta: uma vez que você atribui um valor a uma constante o valor é para ser mantido inalterado ao longo do script. Em todas as circunstâncias onde você razoavelmente prever que o valor original será modificado através do script, você deverá uma variável como seu espaço de armazenamento. Na verdade, você vai usar variáveis ​​na maioria das vezes .

Nesta lição, você vai aprender:

  • como criar variáveis ​​e constantes;
  • como atribuir valores a variáveis ​​e constantes;
  • como usar variáveis ​​em seu código;
  • como nomear variáveis ​​e constantes corretamente.

Como criar variáveis ​​e constantes

Veja como declarar uma variável: 

	  
	    // Para declarar uma variável você usa um nome precedido pela palavra-chave var* /    
	    var amountDue;

No código do exemplo acima, você declarou uma variável chamada amountDue .

A declaração da variável termina com um (;) ponto e vírgula, o que torna uma declaração auto-suficiente.

Além disso, porque o JavaScript diferencia maiúsculas de minúsculas, cada vez que você usar a variável amountDue, certifique-se de manter as maíusculas e minúsculas exatamente como a da declaração original

Veja como declarar uma constante :

	  
	    / * Para declarar uma constante use nome precedido pela  palavra-chave const  
	    tome nota: const não é suportado pelo Internet Explorer, por isto, usar var é mais seguro * /  
	    taxRate const;

No código de exemplo acima, você declarou uma constante taxRate. Como você pode ver, a declaração é muito parecida com a declaração da variável, a única diferença sendo a palavra-chave utilizada: var no caso de uma variável e const no caso de uma constante.

Como atribuir valores a variáveis

Depois de criar uma variável, você deve atribuir (dar) um valor a ela. O sinal de igual (=) é chamado de operador de atribuição : atribuir o valor do que está no lado direito do sinal de = para o que está no lado esquerdo do sinal de =.

Aviso: você não pode realizar operações com variáveis ​​vazias.

Prepare um documento HTML básico com o código JavaScript ilustrado abaixo: 

	      
	  <! DOCTYPE html>      
	  <html>      
	  <head>      
	  <title> Lição 4: Variáveis ​​e constantes </ title>      
	  </ Head>      
	  <body>      
	  <h1> Lição 4: Variáveis ​​e constantes </ h1>    
	  <script type="text/javascript">    
	    // Cria uma variável    
	    var amountDue;    
	    / * Atribuir um valor a ela:    
	    você não sabe o valor ainda    
	    Então, por hora é 0 * /    
	    amountDue = 0;    
	    / * Crie 2 vars mais e atribuir     
	    um valor a cada, ao mesmo tempo * /    
	    var ProductPrice = 5;    
	    var quantidade = 2;    
	    / * Atribuir um valor a amountDue por    
	    multiplicando pela quantidade ProductPrice * /    
	    amountDue = ProductPrice * quantidade;    
	    / * Observe como o valor do amountDue é    
	    alterado de 0 a 10 -    
	    -Alertar o resultado para verificar se é OK    
	    Aviso: você não pode usar ' ou " com o nome var em alerta () * /    
	    alert (amountDue);    
	  </ Script>    	    
	  </ Body>      
	  </ Html>

Você viu a boa e velha caixa de alerta exibindo o número 10? Se você não viu, verifique se o código está digitado exatamente como no trecho acima ( Aviso: Quando o rastreamento bugs (erros) no seu código JavaScript, olhar para os parênteses, ponto e vírgula (;) e aspas ("")).

Isso é ótimo! Experimente com o código acima. Por exemplo, alterar o valor da quantidade e ProductPrice.

Como nomear variáveis ​​(e constantes)

Escolha nomes descritivos para as variáveis ​​( var amountDue; ao invés de var x; ): desta forma o seu código será mais legível e compreensível .
