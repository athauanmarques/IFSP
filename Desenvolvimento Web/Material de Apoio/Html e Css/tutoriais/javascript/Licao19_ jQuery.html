Por que devo usar jQuery?

Aqui estão algumas boas razões para usar jQuery em seus projetos de JavaScript.

jQuery é:

  • extremamente leve (apenas 31 kb);
  • É grátis e muito fácil de incluir em seus projetos: basta baixar a versão mais recente do site do jQuery, ou usar um Content Delivery Network on-line;
  • é continuamente atualizado, mantido e documentado por uma comunidade dedicada de grandes desenvolvedores. Isso garante alta qualidade e apoio na internet;
  • ajuda a superar inconsistências de alguns recursos de JavaScript em diferentes navegadores;
  • e por último mas não menos importante, oferece uma grande variedade de efeitos de animação pré-fabricados que são divertidos de usar.

Como posso incluir jQuery em meu site?

Incluir jQuery em seu projeto é rápido e fácil. Você tem a opção de baixar uma cópia da biblioteca e salvá-lo localmente, ou você pode usar um Content Delivery Network (CDN) para servir o seu site.

Vá para a página de download do site jQuery

Como incluir uma cópia local do jQuery

Se você preferir ter uma cópia local do jQuery:

  1. vá para a página de download e clique na versão reduzida(minified) mais recente do jQuery. No momento em que escrevo, a versão atual é a versão 1.9 ;
  2. copie e cole o código em um arquivo de texto;
  3. salve o arquivo no diretório do projeto web (melhor se você salvá-lo em sua própria subpasta);
  4. em sua página html incluem as apropriadas tags <script>, na seguinte ordem: 
		  
		<! DOCTYPE html>   
		<Html>   
			<Head>   
				<Title> Lição 19: Introdução ao jQuery </ title>  
				 <Script type = "text / javascript" src = "js/latestjquery.js"> </ script>  
				 <Script type = "text / javascript" src = "js/yourscript.js"> </ script>  
			</ Head>   
			<Body>   
				<H1> Lição 19: Introdução ao jQuery </ h1>  
			</ Body>   
		</ Html>

A referência à jQuery vem antes de seu próprio arquivo de código JavaScript. Na verdade, seu código precisa encontrar jQuery totalmente carregado para ser capaz de usá-lo.

Como incluir uma cópia hospedada do jQuery

Se optar por um CDN para servir-se uma cópia do jQuery para o seu site, siga estes passos:

  1. ir para o jQuery CDN e escolha um dos diversos serviços de CDN disponíveis;
  2. copiar a URL do seu CDN de escolha e colá-lo como valor do <script> atributo src da tag , assim: 
     <Script type = "text / javascript" src = "yourCDNUrlAddressjQueryFile.js"> </ script>
  3. colocar uma tag <script> com uma referência ao seu próprio arquivo JavaScript debaixo do jQuery tag <script>, como você fez no exemplo anterior.

É isso aí, está feito!

Como se certificar de jQuery está funcionando

Agora, a biblioteca jQuery pode ser usada em seu projeto. Para certificar-se as coisas funcionam como esperado, crie um arquivo de texto e grave-o como jquery_test.js . Digite o seguinte código: 

		
		 // A maior parte do seu código vai para dentro  
		 // O jQuery   () função de pronto   :  
		 $(document).ready(function ()  
		 {  
		 	alert ("jQuery está trabalhando!");  
		 });  
		 // Certifique-se de manter o controle de chaves, colchetes e ponto e vírgula (;)

Salve todos os seus arquivos e execute a página HTML em um navegador. Se você digitou o código exatamente como o exemplo acima, você deve ser saudado por ... bem, uma caixa de alerta.

O sinal $ é um alias para jQuery. Você poderia substituí-lo com a palavra-chave jQuery e seu código JavaScript-jQuery iria funcionar muito bem. No entanto, menos digitação é o lema do jQuery (e todos os codificadores '), portanto, usar $ é a convenção aceita. 

 .ready (function () {// código JavaScript aqui ...});

é onde a maior parte do seu código JavaScript será colocado. Esta é a maneira inteligente do jQuery saber onde está sendo usado.

Como faço para acessar os elementos DOM usando jQuery?

jQuery agrada tanto web designers porque nos permite selecionar elementos DOM no código tal como fazemos com CSS. Vamos vê-lo em ação.

Selecione os elementos html por id

Lembre-se de nosso velho document.getElementById ()? Bem, jQuery faz seleção por id. Veja como é feito: 

		
		 $(document).ready(function()  
		 {  
		 	var myDiv =   $("# myDiv")   ;  
		 });

O sinal $ na frente de ("#myDiv") pode ser traduzido como: "Hey, browser, conseguir me o elemento com o id de myDiv!"

Como você pode ver, o atributo id é agarrado usando o sinal #, o mesmo que o CSS.

Selecione os elementos html por nome tag

Você também conhece o document.getElementsByTagName (). Este método permite-nos pegar todos os elementos HTML por sua tag.

Veja como jQuery torna isto muito simples: 

		
		 $(document).ready (function)  
		 {  
		 	// Armazena todas as imagens na página em uma matriz  
		 	imagens var =   $("img")   ;  
		 });

Como você pode ver, o $ é mágico.

Selecione os elementos HTML pelo nome da classe

Uma maneira rápida de atingir elementos HTML que compartilham da mesma classe é a seguinte: 

		
		 $(document).ready (function ()  
		 {  
		 	var redElements =   $ (".classRed")   ;  
		 });

Agora redElements contém uma matriz de todos os elementos HTML da classe classRed na página.

Mais uma vez, observe como jQuery facilita a seleção usando a mesma notação que você usa em suas declarações CSS para atingir classes: .className .

Selecione os elementos HTML usando filtros jQuery

jQuery oferece uma sintaxe de filtragem para consultar o DOM que é simples e eficiente.

Aqui está uma lista dos filtros mais comuns:

Filtro Descrição
:first O primeiro item de uma correspondência $ ('p:first') retorna o primeiro parágrafo.
:last O último item de uma correspondência $ ('p:last') retorna o último parágrafo.
:odd Os itens ímpares indexados em um conjunto combinado $ ('tr:odd') retorna linhas ímpares da tabela indexada com 1, 3, etc.
:even Os itens pares indexados em um conjunto combinado $ ('tr:even') retorna linhas pares da tabela indexada à taxa de 0, 2, etc.
:has encontra elementos que têm o elemento filho especificado $ ('p:has (span)') retorna todos os parágrafos que contêm um elemento de span.
:eq retorna o elemento com o índice combinado $ ('p:eq (1)') retorna o segundo parágrafo começando a contar de 0.
:contains(x) Retorna todos os elementos que contém o texto x $ ('div:contains(foo)') tem como alvo todos os divs com o texto 'foo'.

A lista completa de todos os métodos de seleção e de filtros estão disponíveis no website jQuery . Dê uma olhada e fazer experiências com as amostras de código disponíveis para ganhar mais familiaridade com a nova abordagem.

Como posso atribuir manipuladores de eventos usando jQuery?

Os manipuladores de eventos são blocos de código, geralmente funções, que especificam o que sua aplicação deve fazer quando um evento específico, como, por exemplo, um clique de botão, ocorre.

Até agora, você já usou duas abordagens:

  1. atribuir uma função para o próprio elemento html : <type = "button" input onclick = "doSomething ()" />
  2. atribuir uma função a propriedade adequada do objeto DOM : myElement.onclick = doSomething ;

Ambas as abordagens são boas. No entanto, a primeira peca contra o princípio da separação de interesses . Na verdade, ele mistura código JavaScript com código HTML.

A segunda abordagem está em conformidade com o princípio da separação de interesses, mas limitada no caso de desejarmos anexar mais de função para o mesmo elemento.

Na verdade, há uma terceira abordagem que supera todas as deficiências, mas com uma restrição: ele é implementado de forma diferente em navegadores Internet Explorer em relação a todos os outros principais navegadores.

A boa notícia é: jQuery está aqui para fazer tratamento de evento rápido e fácil.

O método bind ()

bind (eventType, manipulador) funciona da seguinte maneira.

Suponha que você tenha um elemento html com um id myElement , e uma função chamada sayHello() . Você quer disparar sayHello() quando o usuário clicar no elemento html. Isto é como você conseguir isso usando bind (): 

		
		 $(document).ready (function ()  
		 {  
		 	// Cria a função sayHello  
		 	function sayHello()  
		 	{  
		 		alert ("Olá jQuery");  
		 	}  
		 	// Anexar o manipulador usando .bind ():  
		 	myElement   .bind ('click', sayHello)   ;  
		 });

Você pode encontrar mais detalhes sobre o método bind () em http://api.jquery.com/bind/ .

A abordagem ready-made

jQuery oferece alguns manipuladores ready-made correspondentes aos eventos de JavaScript. Aqui está uma lista dos mais utilizados:

Manipulador Descrição
click() Vincula um manipulador a um evento onclick
hover() Vincula um manipulador para os eventos onmouseover e onmouseout
change() Vincula um manipulador para o evento onchange (quando o conteúdo de um campo de mudanças).
select() Vincula um manipulador para o evento onselect (quando o texto for selecionado).
submit() Vincula um manipulador para o onsubmit caso de um elemento de formulário.
focus() Vincula um manipulador para o evento onfocus (quando um elemento recebe foco).
keypress() Vincula um manipulador para o evento onkeypress (quando uma tecla no teclado do computador é pressionado).

Supondo que você tenha um elemento html com um atributo id de myElement , e uma função chamada sayHello() - como no exemplo anterior - você pode vincular sayHello () para o evento onclick de myElement dentro função pronto do jQuery () da seguinte forma: 

		
		 myElement.click (sayHello)   ;

O método on ()

O método on () pode ser usado para anexar manipuladores para ambos os elementos já presentes na página HTML desde o início e em elementos adicionados dinamicamente (ao contrário de bind()). Veja como ele é usado.

Usando a mesma função sayHello () , dentro do jQuery .ready ): 

		
		 myElement. on ('click', sayHello)   ;

Mais detalhes sobre o método on () pode ser encontrado em http://api.jquery.com/on/ .

Funções anônimas

Amostras de código usando jQuery muitas vezes empregam funções anônimas . Estas são funções sem um nome, e pode ser bastante útil, apesar de eu achar nomes mais útil em termos de legibilidade do código. Se substituirmos o sayHello() por uma função anônima, temos:

		
		 // Usando bind ():  
		 myElement.bind ('click',   function ()  
		 {  
		 	alert ("Olá jQuery");  
		 });  
		 / ******************************** /  
		 // Usando click (), temos:  
		 myElement.   click (function ()  
		 {  
		 	alert ("Olá jQuery");  
		 });  
		 / ************************************** /  
		 // Usando on (), temos:  
		 myElement.on ('click',   function ()  
		 {  
		 	alert ("Olá jQuery");  
		 });

Adicionar e remover elementos DOM

Adicione novos elementos DOM

Você pode recuperar os elementos HTML e adicionar novos elementos html para a página usando jQuery .html () método. Quando você não passar quaisquer argumentos, html () recupera um elemento DOM, se você passar um argumento de cadeia, html () acrescenta que seqüência de caracteres para a página.

Se você só precisa adicionar novo texto dentro de um elemento HTML, você pode usar do jQuery .text () método e passar um argumento string representando o texto que pretende adicionar. Como .html (), você pode usar .text () sem passar quaisquer argumentos para recuperar o conteúdo do texto a partir da página HTML.

Para testar .html () preparar uma página HTML contendo uma tag <div> com um id myElement , a tag <p> e algum texto. Primeiro vamos recuperar o elemento parágrafo existente, então vamos criar um novo parágrafo. 

		
		 $ (Document) .ready (function ()  
		 {  
		 	// Use html () para recuperar o conteúdo:  
		 	. var pageParagraph = $ ('p') html ();  
		 	// O alerta deve exibir <p> 's conteúdo  
		 	alert (pageParagraph);  
		 });  
		 / ************************************************* * /  
		 // Agora vamos mudar o conteúdo da div  
		 // Adicionando um novo parágrafo.  
		 	. var newParagraph = $ ("# myElement ') html (' <p> Olá novo parágrafo </ p>! ');  
		 // Fechar todos os seus suportes:  
		 });

Salve todos os seus arquivos e visualize o resultado em um navegador.O alerta deve exibir o conteúdo do parágrafo original, e, posteriormente, a página deve exibir o conteúdo do novo parágrafo.

A maneira como você implementar .text() é o mesmo que .html(). Experimente! Mais detalhes sobre .html() podem ser encontrados no http://api.jquery.com/html/ , e em .text () pode ser encontrado no http://api.jquery.com/text/ .

Remover elementos DOM

Uma maneira simples de fazer isso é com o método remove () do jQuery.

Para testar, use a mesma página HTML do exemplo anterior e adicione um elemento de botão com um id btnDelete .

Em um novo arquivo JavaScript, adicione o seguinte código: 

		
		 $(document).ready (function ()  
		 {  
		 	// Usar um manipulador de clique e uma função anônima  
		 	// Para fazer o trabalho  
		 	$ ("# BtnDelete ').Click(function ()  
		 	{  
		 		// Apagar todos os elementos <p>  
		 		$ ('P') remove ().   ;  
		 	});  
		 });

Salve todos os seus arquivos e visualize a página HTML em um navegador. Clique no botão e ver os números na página de desaparecer.

Mais sobre .remove () pode ser encontrado em http://api.jquery.com/remove/ .
