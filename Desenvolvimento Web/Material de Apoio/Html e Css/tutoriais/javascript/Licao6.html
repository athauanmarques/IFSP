Lição 6: Comandos de Seleção

Agora que o nosso código JavaScript pode calcular, comparar e determinar as condições verdadeiras e falsas , as coisas começam a ficar mais interessantes.

Nesta lição você vai injetar o poder de escolha em seus scripts com declarações condicionais . Estas declarações permitir que o seu script para avaliar as condições dadas na base dos quais para executar uma ou mais ações. Mais especificamente, você vai aprender como usar:

  • if
  • if ... else
  • if ... else if ... else
  • switch ... case

No processo, você também vai aprender como usar atalhos de operadores de atribuição.

if

A palavra-chave if é usada em JavaScript para executar um teste básico para a verificar a verdade ou falsidade de uma determinada condição e executar um pedaço de código apropriado. Aqui está a sua estrutura: 

	
	    // Se a condição em parênteses seja verdadeira    
	  
	    if (     condição     )    
	  
	    {    
	  
	    // Fazer algumas coisas aqui    
	  
	    }

Vamos traduzir o código

O if acima diz:

"Hey navegador, se a condição de nos parênteses é verdadeira, execute os comandos entre chaves. Se a condição não for verdadeira, então ignorar tudo e seguir em frente!"

Importante: não esquecer o parêntesis () ou as chaves {} .

if ... else

Use esta instrução para executar um comando se a condição for verdadeira e outro comando se a condição for falsa . Aqui está a sua estrutura: 

	
	    // Se a condição for verdadeira    
	  
	    if (     condição     )    
	  
	    {    
	  
	    // Fazer essas coisas    
	  
	    }    
	  
	    // Caso contrário    
	  
	    else    
	  
	    {    
	  
	    // Fazer essas outras coisas    
	  
	    }

if ... else if ... else

Use essa declaração se você quiser que o seu código avalie várias condições para executar os comandos apropriados. Aqui está a sua estrutura: 

	
	    // Se a condição for verdadeira    
	  
	    if (     condição     )    
	  
	    {    
	  
	    // Fazer essas coisas    
	  
	    }    
	  
	    // Se esta outra condição é verdadeira, em vez    
	  
	    else if (     condição     )    
	  
	    {    
	  
	    // Fazer essas outras coisas    
	  
	    }    
	  
	    // Se nenhuma das condições acima é verdadeira    
	  
	    else    
	  
	    {    
	  
	    // Fazer isso    
	  
	    }

switch...case

Em alternativa a uma declaração if ... else if ... else você pode usar uma instrução switch . Aqui está a sua estrutura básica: 

	
	    // Condição para avaliar    
	  
	    switch (condição)    
	  
	    {    
	  
	    //Neste caso, fazer    
	  
	    case 1:    
	  
	    executar um bloco de código    
	  
	    // Parar por aqui: não há necessidade de continuar    
	  
	    break;    
	  
	    // Neste outro caso fazer   
	  
	    case 2:    
	  
	    executar bloco de código 2    
	  
	    // Parar por aqui: não há necessidade de continuar    
	  
	    break;    
	  
	    // Caso nenhuma das condições anteriores    
	  
	    default:    
	  
	    código para executar quando todas as condições anteriores resultaram false    
	  
	    }

Operadores de atalho

Quando você adicionar vários valores a uma variável, um operador de atalho é muito útil.

A expressão x += 2 é o mesmo que x = x + 2 .

Como se pode ver, o valor de x é tomado como o ponto de partida, 2 é adicionado a ele, e o valor resultante é transferido para x. Usar um operador de atalho poupa de ter que redigitar x.

Quando aplicado ao trecho de código acima, a expressão infotext += "em vez de US $" é o mesmo que infotext = infotext + "em vez de US $" . Se nós simplesmente usar um operador de atribuição (=), neste caso, o texto já armazenado na infotext serão eliminados pelo novo valor. Mas este não é o que queremos neste script.

Atalhos úteis também estão disponíveis para todos os outros operadores matemáticos:

  • -=
  • *=
  • /=
  • %=
