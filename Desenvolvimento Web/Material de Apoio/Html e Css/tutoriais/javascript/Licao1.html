Lição 1: O que é JavaScript?

Até o final desta lição, você vai saber:

  • a diferença entre JavaScript e Java;
  • o que JavaScript pode e não pode fazer.

JavaScript e Java são a mesma coisa?

Não, eles não são .

Java (desenvolvida pela Sun Microsystems) é uma poderosa linguagem de programação e muito mais complexa na mesma categoria como C e C + +.

JavaScript foi criada por Brendan Eich a Netscape e foi introduzido pela primeira vez em Dezembro de 1995 sob o nome de LiveScript . No entanto, foi rapidamente rebatizada JavaScript, embora o nome oficial do JavaScript seja ECMAScript , que é desenvolvido e mantido pela ECMA (European Association fabricante do computador) organização Internacional.

JavaScript é uma linguagem de script, ou seja, uma linguagem de programação leve, que é interpretado pelo mecanismo do navegador quando a página é carregada.

Coisas que você não pode fazer com JavaScript

Você não pode forçar o JavaScript em um navegador.

JavaScript é executado no cliente , isto é, no Brower. Se você usar um navegador mais antigo, sem suporte a JavaScript, ou se você simplesmente escolher para desativar o JavaScript no seu navegador, em seguida, um script JavaScript não pode trabalhar.

Conclusão: ao contrário do que acontece com as línguagens que são executados no servidor , como PHP, você nunca conhecer plenamente com certeza o impacto que o navegador visitantes do seu site vão utilizar terá em seu script, ou se os visitantes vão optar por ativar o JavaScript apoiar off.

Você não pode acessar ou alterar os recursos de outro domínio de internet com JavaScript.

Isso é chamado de Política de mesma origem . Bem, como você gostaria que se de repente todos os comentários simpáticos seus visitantes deixados em seu site começaram a desaparecer, ou mudar de lugar em sua página por causa de um script JavaScript safada sendo executado em outro site?

Este é exatamente o tipo de situação desagradável que política de mesma origem é projetado para impedir. Conclusão: o script JavaScript só podem acessar recursos em seu site.

Você não pode acessar os recursos do servidor com JavaScript.

Devido JavaScript é uma linguagem do lado do cliente, é limitado ao que pode ser feito no cliente, que é, geralmente no ambiente do navegador. Um script JavaScript não podem acessar recursos de servidor, como bancos de dados.

Coisas que você pode fazer com JavaScript

Com JavaScript você pode:

Colocar o texto em uma página HTML on-the-fly.

Digamos que você queira exibir uma mensagem de agradecimento agradável para um usuário que acaba de apresentar um formulário de comentário em seu site. Obviamente, a mensagem pode ser adicionada após o utilizador tem a forma apresentada.

Você poderia deixar o servidor fazer isso. No entanto, se o site é muito ocupado e seu servidor processa centenas de formulários por dia, pode demorar um pouco para a sua mensagem de agradecimento a aparecer para o usuário.

Aqui está o JavaScript para o resgate. Porque o JavaScript é executado no navegador do usuário, o bilhete de agradecimento pode ser adicionada e exibida na página quase instantaneamente, tornando seus usuários do website feliz.

Faça as suas páginas de resposta.

Ambientes Web são dinâmicos, as coisas acontecem o tempo todo: as cargas de páginas da web no navegador, o usuário clica em um botão ou mover o mouse sobre um link, etc Estes são chamados de eventos (que será o tema da lição 3).

Com JavaScript você pode fazer a página ragir imediatamente a esses eventos da maneira que você escolher: por exemplo, mostrando ou escondendo elementos de página específicos, alterando a cor de fundo, etc

Detectar browsers dos visitantes.

Você pode usar um script JavaScript para detectar o navegador do visitante, ou, melhor ainda, você pode detectar o que caracteriza um determinado navegador ou não apoiar. Dependendo do navegador e seus recursos, você pode optar por carregar uma página, concebidas especificamente para esse tipo de browser (lição 14).

Criar cookies.

Um script JavaScript é ótimo se você quiser criar cookies para que os visitantes podem desfrutar de uma experiência personalizada na próxima vez que visitar o seu site (lição 15).

Validar dados de formulário web.

Você pode usar um script JavaScript para validar os dados do formulário antes de o formulário é enviado para um servidor. Isso evita que o servidor de processamento extra (lição 16).

E muito ... muito mais.

Aprendizagem JavaScript permitirá que você adicionar efeitos de animação divertidos para suas páginas da web sem usar um flash externo plug-in, use os novos recursos do HTML5, como lona (para desenhar diretamente na sua página web) e arrastar e soltar as capacidades, integrar o seu site com serviços web externos, tais como Facebook, Twitter, etc
