Tutorial PHP - Tabela de conteúdos

  • Introdução
    Uma breve introdução ao tutorial e o que você aprenderá.
  • Lição 1: O que é PHP
    Breve relato de como funciona o PHP, o que significa dizer que PHP é uma tecnologia do lado do servidor e um resumo dos assuntos que você aprenderá nas próximas lições.
  • Lição 2: Servidores
    As diferentes opções para fazer o PHP funcionar no seu computador ou em um servidor web.
  • Lição 3: Sua primeira página PHP
    Nesta lição você criará sua primeira página PHP. Aprenderá também uma maneira de verificar se o seu servidor está em condições de processar PHP.
  • Lição 4: Trabalhando com datas e horas
    Introdução às funções para trabalhar com datas e horas.
  • Lição 5: Loops
    Loops se destinam a executar trechos de scripts repetidamente. Nesta lição estudaremos os loops, tais como, while e for.
  • Lição 6: Condicionais
    Condicionais são usadas para controlar a execução de um script PHP. Estudaremos as condicionais if ... elseif ... else... e switch ... case.
  • Lição 7: Insira comentários nos seus scripts
    Comentários se destinam a tornar seus scripts PHP mais claros e fáceis de entender. Eles se constituem em um valioso auxiliar se você ou outra pessoa resolver introduzir mudanças no código futuramente.
  • Lição 8: Arrays
    Nesta lição você aprenderá o que é array, para que servem e como usá-los.
  • Lição 9: Funções
    Nas lições anteriores você aprendeu como usar algumas funções prontas do PHP. Agora você aprenderá como criar suas próprias funções.
  • Lição 10: Passando variáveis no URL
    Aprenda como passar variáveis e valores de uma página para outra usando HTTP query string.
  • Lição 11: Passando variáveis de um formulário
    Sites interativos requerem coleta de dados do usuário. Uma das formas mais comuns de coletar dados é com uso de formulários.
  • Lição 12: Sessões
    Sessões são usadas para armazenar e recuperar informações durante o tempo em que o usuário permanece no site.
  • Lição 13: Cookies
    Cookies são usados para armazenar e recuperar informações sobre o usuário a serem usadas em cada visita ao site.
  • Lição 14: Sistema de arquivos
    Denomina-se sistema de arquivo a um conjunto de funcionalidades para acessar os arquivos do servidor. As funcionalidades permitem manipular arquivos, diretórios e drivers no servidor usando scripts PHP.
  • Lição 15: Ler arquivos de texto
    Nesta lição usaremos o sistema de arquivo para proceder a leitura de um arquivo de texto no servidor. Arquivos de texto são úteis para armazenamento da dados de vários tipos.
  • Lição 16: Escrever em arquivos de texto
    Nesta lição estudaremos como escrever em um arquivo de texto usando o sistema de arquivo. Arquivos de texto são úteis para armazenamento da dados de vários tipos.
  • Lição 17: Banco de Dados
    Neste tutorial usaremos o banco de dados MySQL. MySQL é o caminho natural para se começar quando se pretende usar banco dados com PHP.
  • Lição 18: Criando Banco de Dados e suas Tabelas
    Nesta lição abordaremos dois métodos para criação de banco de dados e tabelas. Primeiro veremos como criá-las com uso de PHP e depois como uso de uma ferramenta amigável ao usuário chamada PhpMyAdmin.
  • Lição 19: Inserindo dados em um Banco de Dados
    Nesta lição aprenderemos como usar declarações SQL para inserir dados em um banco de dados. Veremos ainda os diferentes tipos de dados e comentaremos sobre alguns erros comuns cometidos quando da inserção de dados no banco de dados.
  • Lição 20: Extraindo dados de um Banco de Dados
    Nesta lição aprenderemos como usar declarações SQL para extrair dados de um banco de dados.
  • Lição 21: Apagando dados de um Banco de Dados
    Nesta lição aprenderemos como usar declarações SQL para apagar dados de um banco de dados.
  • Lição 22: Atualizando dados de um Banco de Dados
    Nesta lição aprenderemos como usar declarações SQL para atualizar dados em um banco de Dados.
